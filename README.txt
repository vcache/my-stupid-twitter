1. Dependencies
	- simplejson
	- oauth2
	- your own 'consumer key' and 'consumer secret'
	- maybe something else
	- oh, also python

2. Usage
	1) Run:
	$ python ./my-stupid-twitter.py

	2) Answer questions

	Keys:
	x - close application
	left/right arrows - change link
	enter - open link in the default browser
	spacebar - draw funny yellow line
